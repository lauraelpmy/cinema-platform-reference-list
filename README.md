# cinema-platform-reference-list
本项目致力于构建一个影视类网站的参考索引列表，涵盖电影、电视剧、综艺及动漫等相关站点的公开信息。仓库通过简要说明站点定位与内容范围，帮助使用者快速了解不同影视平台的基本情况。所有数据均来源于互联网公开渠道，仅用于资料汇总与信息参考，不包含任何在线播放资源或第三方接口。该仓库不参与站点推广、不涉及商业合作，仅作为影视网站生态的长期记录与观察样本。
